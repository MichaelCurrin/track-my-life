## Run the app

Start the CherryPy server, to serve HTML and allow REST API requests

1. Tail your log files (_to be implemented_)

    ```
    $ cd trackMyLife/app/var/trackMyLife
    $ tail -F *.log
    ```

2. Start the app

    ```bash
    $ cd trackMyLife/app
    $ ./trackMyLife.py
    ```

3. Open in the browser
  
  - go to http://localhost:8082
